# 🚀 Cyberbullying Detection System (Hinglish NLP)

📌 **Overview**
This project is a Machine Learning based Cyberbullying Detection System** designed to identify harmful and abusive text in Hinglish (Hindi-English mixed language).

It uses **Natural Language Processing (NLP)** techniques along with multiple classification models to accurately detect cyberbullying content in real-time.

----------------------------------------------------------------------------------------------------------------------------------------------------------

🎯 **Key Features**
🔍 Detects cyberbullying in Hinglish text
 🤖 Multiple ML models implemented and compared
 ⚡ Real-time prediction system
 💬 GUI-based Safe Chat application
 📊 TF-IDF feature extraction
 🧠 Model performance comparison

-------------------------------------------------------------------------

🛠️ **Tech Stack**
Programming: Python  
Machine Learning: Scikit-learn  
NLP: TF-IDF, Text Preprocessing  
GUI: Python Tkinter  
Dataset: Custom Hinglish Dataset  

------------------------------------------------------------------------------

🧠 **Machine Learning Models Used**
Logistic Regression  
Random Forest  
Decision Tree  
Support Vector Machine (SVM)  
Naive Bayes  
K-Nearest Neighbors  
AdaBoost & Bagging  

--------------------------------------------------------------------------------
⚙️** How It Works**
1. Text input is given by user  
2. Preprocessing (stopwords removal, cleaning)  
3. TF-IDF feature extraction  
4. Model predicts
